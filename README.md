# Ride Sharing System

A simple ride-sharing app built in C++ and Smalltalk to show how encapsulation, inheritance, and polymorphism work in practice.

## Features

- Three ride types (Base, Standard, Premium) with different per-mile rates
- Driver and Rider classes with private data and controlled access
- Same method calls produce different results depending on ride type

## How to Run

### C++

```bash
g++ cpp/main.cpp -o cpp/main.exe
./cpp/main.exe
```

### Smalltalk

Paste the contents of `smalltalk/RideSharingSystem.st` into [JDoodle Smalltalk](https://www.jdoodle.com/compile-smalltalk-online/) and run it.
